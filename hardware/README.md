## Contribution notes
- Keep instructions reproducible on macOS first; add Windows/Linux notes when tested.
- Prefer simple BOMs and wiring tables; link to firmware repos or sketches kept in this folder.
- Record tested firmware commits and flashing commands so they can be repeated later.
